Lantern, Social Media Poster

By: Matthew Hatch, Chad Eddington, and Kyler Hinton

Have you ever wanted to be able to post to separate social media sites with the same post? Well now you can!

Our app takes 3 different social media sites and allows the user to type 1 post and have it send that post to all 3 separate sites


The user simply types in the message box their social media update and on "Post" the post will be sent to the 3 separate accounts!

Not only is this clever but it is also convenient! It is as easy as a of a button!
